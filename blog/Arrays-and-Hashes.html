Hashes and Arrays

What is an array ?

An array is a data structure that is essentially just an ordered list that can be used to store data of any type. What this means is that i can store integers, floats, strings, booleans and even other arrays. Ruby being such a flexible language alows you to define an array several ways here are few examples. 


arr = ["hello", 1, 9.0, nil, true]
new_arr = %w[the cow jumped over the moon]
other_arr = []

Accessing elements

Each object that is stored in an array is called an element. An array is indexed which means each element is stored a specific number. To access the elelment you type the array name plus the number between []. It is important to note that arrays indices start with 0, here is an example. 


arr = [1, 2, 3, 4, 5]
arr[0] #> 1
arr[4] #> 5
arr[2] #> 3

Useful built in methods

Apart from creating and adding elements to an array . Ruby has a vast selection of powerful methods that can manipulate the data in arrays. Note that some methods permanently change the initial array that it is called upon while some don't, in order to know the methods that don't permanently change the array, the best method is to just experiment in irb. 

arr = [how, to, read, a, book]
arr.pop #> "book" // removes the last elemennt of the array indices 
arr.delete("a") // removes all instances of the element provided
arr << "a" // adds the string a back to the array
array.push("book") // adds the string book to the array

What is a hash ?

A hash is a data-structure that stores items in a collection of key-value pairs. The difference between the two is that unlike arrays that store items through an ordered index, hashes allows you to use any object type. All key-value pairs in a hash are surrounded by curly braces {} and comma separated. Here is a simple way to create a hash using symbols as keyswith multiple key value pairs 

 fruits_color = { orange: "orange" , grape: "purple" , banana: "yellow" , apple: red}

Adding, deleting and retrieving data 

fruit_color[:pear] = green // adds new key value pair to hash
fruit_color.delete[:orange] // removes the key value pair of (orange: "orange")
fruit_color[:apple] // outputs the value ok key :apple

To sum it up, hashes and arrays are two powerful data structures that can be usesd to model data in either an index or associative pattern. As you grow as a developer and with more practice you will know which one to apply to solve the problem at hand.
