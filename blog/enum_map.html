Enumerable#map

Enumerables are a basically packaged up methods that are mixed in with classes that implement their own each method.The Enumerable module provides a set of methods to traverse, search, sort and manipulate collections. In this post we will look at the enumerable method map which is mixed in with Array and Hashes classes.But first lets look at each to understand the difference. 

[1, 2, 3, 4, 5].each { |element| puts element + 1 } 
3
4
5
6
=> [1, 2, 3, 4]

As you can see above each iterates through the array and passes the block on each value but at the end it discards the changes and returns the original array. 

[1, 2, 3, 4, 5,].map {|element| element ** 2 }

=> [1, 4, 9, 16, 25]

However, as seen above like the each when given a block it invokes the given block once for each element in the array. The difference is that map creates a new array with the values returned from the block. So to sum up each and map are two iterators that both traverse a collection of objects. But the key difference to remember is each always returns the original while map transforms the collection.
