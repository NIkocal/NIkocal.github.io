Looping in Javascript and Ruby

I've spent a couple months learning Ruby but now i am focusing on getting comfortable with Javascript, which is not as hard to learn becasue it will be my second language. In this post i will demonstrate how to use a for loop through 1-100 in both Ruby and Javascript. To make it fun i will demonstrate the FizzBuzz problem in both languages. I will being with the Javascript axample first. 

  

for (var i=1; i <= 20; i++)
{
    if (i % 15 == 0)
        console.log("FizzBuzz");
    else if (i % 3 == 0)
        console.log("Fizz");
    else if (i % 5 == 0)
        console.log("Buzz");
    else
        console.log(i);
}

Here is the same problem in Ruby

            
            
    1.upto(100) do |i|
